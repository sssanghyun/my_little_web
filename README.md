# my_little_web
개인 포트폴리오를 정리할 웹페이지가 필요하여 개발한 반응형 웹페이지 입니다.
개발부터 배포까지 진행하였고 과정은 아래와 같습니다.
각 페이지 초안 개발 → 반응형 적용 → 리팩터링 → 도메인 구매 → AWS Route53 호스팅 등록 
→ 가정용 NAS(시놀로지) 내 도커/ubuntu20.04 환경에 배포 → SSL 인증서 적용

### 개발 환경
- windows 11(개발)
- docker ubuntu 20.04(배포)
- Next.js(v14.0.2)
- Express(v4.18.2)
- Node.js(v20.9.0)
